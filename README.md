# Expense_Tracker_Project
A simple console based Python project for tracking and summarizing personal expenses.
# Features
Add Expenses — record amount, category, and date,
View All Expenses — see a complete list of saved entries,
Summarize Spending — view total expenses by category,
Data Storage — expenses saved in data/expenses.txt,
Persistent Data — keeps records even after program exit,
Modular Code Design — organized using functions and file handling,
